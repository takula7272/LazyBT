cmd line: python3 lazyBT.py -d <log_path> -n <partial_device_name> -m <partial_mac> -r -t <profile>

-d specific the parsing log path, default is the rootdir of lazyBT.py
-n dump all device which device name contain <partial_device_name>
-m dump all device which device mac contain <partial_mac>
-r parsing device from log under <log_path>, default dump the alldevice.txt file which is generated by parsing all log in first time
-t specific profile log parsing, now support A2DP connection and A2DP audio state only

if no -n and -m option, all devices will be dumped
